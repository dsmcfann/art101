Lab 15 - AJAX

Challenge

For this lab, we needed to utilize our new knowledge of AJAX and APIs to connect an already existing API to our website. I initially wanted to add the PokeAPI to my website, which I though would be fun given my love of the Pokemon franchise. Eventually, I decided to switch to a different API based on Dungeons and Dragons spells.

Problems

Despite having the example of the successful lab from Mason, who showcased how to make the lab function using the "YesNo" API, I found quite a bit of difficulty making the lab function the way it was supposed to. I also could not figure out how to output the JSON as HTML onto the page. However, I eventually made it work using a different API and also through some trial and error.

Reflection

Though I had quite a few problems in the creation of this lab, making these errors allowed me to learn even more about how APIs and AJAX worked.
